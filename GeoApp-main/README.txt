This app accesses the following API Endpoints from GeoDB:

Countries - url: 'https://wft-geo-db.p.rapidapi.com/v1/geo/cities',
Country Details: ''https://wft-geo-db.p.rapidapi.com/v1/geo/countries/<countryID>'
Cities: 'https://wft-geo-db.p.rapidapi.com/v1/geo/cities'

See screenshots in folder "Screenshots" demonstrating functionality per assignment requirements

Use this app to

---Find the capital & calling code of a country using the country's ISO3166 Code (i.e. US)

---Add countries to local SQLite database and view your countries's data 

---Filter your countries by the name of one of its cities

---Filter your countries by Currency Code
